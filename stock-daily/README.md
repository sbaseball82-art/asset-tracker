# 📈 毎日自動更新・株式マーケット情報画像ジェネレーター

毎朝7:30(日本時間)に、日米の株式市場の有益情報を1枚の画像にまとめて自動生成します。
生成された画像とX投稿用コメントをコピーして、そのままXに投稿できます。

## 画像に含まれる内容

1. 主要指数・為替の前日比(日経平均 / TOPIX / ドル円 / NYダウ / S&P500 / NASDAQ / FANG+指数 / 半導体ETF(SMH) / DRAM関連平均)
   - DRAM関連平均は Micron・SanDisk・WesternDigital・キオクシア 4社の前日比の単純平均(自作指標・概算)
   - #FANG / #DRAM のハッシュタグは、その日の主役(値上がり1位)が該当銘柄のときだけ自動付与
2. 値上がり率TOP5(日本の主力株40銘柄・米国の主力株40銘柄から自動選出)
3. 注目ニュース見出し(NHK経済・CNBC・Yahoo Finance)
4. 今後の注目イベント(日銀会合・FOMC・米雇用統計・メジャーSQ ※2026年公式日程)

## 🚀 導入手順(初回のみ・約5分)

このプロジェクトは `asset-tracker` リポジトリの `stock-daily/` フォルダに組み込み済みです。
ワークフローはリポジトリ直下の `.github/workflows/daily_report.yml` に配置されています。
残りの手順は2つだけ:

1. **Actionsの書き込み権限を有効化**(既に有効なら不要)
   - リポジトリの「Settings」→「Actions」→「General」
   - 一番下の「Workflow permissions」で **「Read and write permissions」** を選択して保存
2. **動作テスト(手動実行)**
   - 「Actions」タブ →「daily-stock-report」→ 右側の「Run workflow」ボタン
   - 2〜3分後に緑のチェック✅が付けば成功

## 📱 毎日の使い方(30秒)

1. スマホでリポジトリの `stock-daily/output/latest.png` を開く → 画像を保存
2. `stock-daily/output/post_text.txt` を開く → コメントをコピー
3. Xに貼り付けて投稿するだけ

- 実行時刻: 毎朝 **7:30 JST**(米国市場クローズ後なので前夜の米国結果が反映されます)
- 過去30日分の画像は `stock-daily/output/report_日付.png` として自動保存(古い分は自動削除)

## ⚙️ カスタマイズ

| 変更したい事 | 場所 |
|---|---|
| 実行時刻 | `daily_report.yml` の `cron: '30 22 * * *'`(UTC表記。JST−9時間) |
| 対象銘柄 | `daily_report.py` の `JP_UNIVERSE` / `US_UNIVERSE` |
| ハッシュタグ・投稿文 | `daily_report.py` の `make_post_text()` |
| 画像の色 | `daily_report.py` の `BG` / `ACCENT` / `UP` / `DOWN` |

## 🛡 止まらないための仕組み(実装済み)

- データ取得は各3回リトライ。一部が失敗してもそのセクションだけ「取得失敗」と表示し、画像は必ず生成
- スクリプト全体もGitHub Actions側で最大3回再実行
- 経済カレンダーはネット不要(2026年の公式日程を内蔵)なので絶対に止まらない
- 全データソースが全滅した場合のみ失敗扱い(Actionsのメールで通知されます)

## ⚠️ 注意

- GitHub Actionsの無料枠(Privateリポジトリで月2,000分)に対し、本ワークフローは月90分程度しか使いません
- スケジュール実行は数分〜数十分遅延することがあります(GitHubの仕様)
- 60日間リポジトリに一切変更がないとスケジュールが自動停止します(毎日コミットされるため通常は問題ありません)
- データはYahoo Finance等の無料ソースです。表示内容は情報提供目的であり、投資助言ではありません
- 2027年以降は `EVENTS_2026` に翌年の日程を追記してください(毎年12月頃にClaudeに頼めば更新できます)
